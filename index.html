Helium Rewards Checker is a Helium community app built to check the HNT rewards of any miner of the Helium Blockchain using a predefined time period or a custom one. It also allows you to see the value of the rewards in different fiat currencies and other relevant information (total rewards since a miner was added, date it was added, average rewards per day / hour within the selected period, miner's and owner's address, location, etc.).

Another reason that brought me to develop this tool is to provide realtime information about the HNT rewards. The Helium Explorer and the Helium Hotspot App (to this date) are not only quite slow. They also do not give you realtime information (at least not the most up-to-date from its blockchain).

Essentially this webapp uses Helium's Blockchain API to retrieve the desired information directly from the Helium's Blockchain, combined with the current HNT fiat value via Coingecko's API. In case you have any suggestions or find any bugs, please write me at:
soacmaster[at]proton.me

Last but not least, keep in mind that this app is free (and it wants to stay as such: no ads, no annoyances). Currently this site is having around 1000 unique visits a month. If on average every user would donate at least 0.1 HNT, I could not only cover the hosting, domain, developing and mantaining costs, but it would also allow me to improve it a lot and develop an Android and iOS app out of it. So, if you think this app is useful, please use the QR code to donate HNT to support it.
