Helium Rewards Checker is an unofficial app built to check the HNT rewards of any miner of the Helium Blockchain using a predefined time period or a custom one. It also allows you to see the value of the rewards in different fiat currencies and other relevant information (total rewards since a miner was added, date it was added, average rewards per day / hour within the selected period, miner's and owner's address, location, etc.).

Another reason that brought me to develop this tool is to provide realtime information about the HNT rewards. The Helium Explorer and the Helium Hotspot App (to this date) are not only quite slow, they also do not give you realtime information (at least not the most up-to-date from Helium's blockchain).

Essentially this webapp uses Helium's Blockchain API to retrieve the desired information directly from Helium's Blockchain, combined with the current HNT fiat value via Coingecko's API. Also, since you can access all the code (all non-ofuscated neither minimized frontend JavaScript running in your browser 😉) feel free to revise it, send me suggestions or report bugs if you find any:
soacmaster[at]proton.me

Last but not least, keep in mind that this app is free (and it wants to stay as such: no ads, no annoyances). However, the effort, the hosting, the domain, the maintenance, etc., are not. So, if you think this app is useful, please use the QR code to donate HNT to support it.
